# WalletGuard: Smart Financial Companion

A mobile application I developed to help users manage their finances more effectively through intelligent automation and data-driven insights.

## 🎯 Project Origin

I created WalletGuard after noticing how many people struggle with tracking their daily expenses and making informed financial decisions. My goal was to leverage my background in mobile development and machine learning to build something practical that could genuinely help people improve their financial habits.

## 🏗️ System Design

### Mobile App (Flutter)
- **Cross-platform development** using Flutter for iOS and Android
- **Cloud backend** via Firebase for user data and authentication
- **Modern UI** with Material Design components and smooth animations
- **Local storage** using SharedPreferences for offline functionality

### ML Backend (Python)
- **Web API** built with Flask for mobile-backend communication
- **Predictive models** using Random Forest and Naive Bayes algorithms
- **Image processing** with OCR for automatic receipt data extraction
- **Data handling** with pandas for financial data analysis

## 🚀 What It Does

### Everyday Features
- **Secure login system** with email authentication
- **Financial overview** showing income, expenses, and savings trends
- **Expense tracking** with categories and detailed records
- **Budget monitoring** with visual progress indicators
- **Savings goals** with milestone tracking

### Smart Features
- **Camera receipt scanning** that automatically extracts purchase data
- **Spending predictions** based on historical behavior patterns
- **Payment reminders** for bills and subscriptions
- **Email reports** with weekly financial summaries
- **Interactive charts** for visualizing financial data

## 🔧 Getting Started

### What You'll Need
- Flutter SDK version 3.5.4 or newer
- Python 3.x with package manager (pip)
- Firebase project setup with authentication enabled
- Mobile development environment (Android Studio or Xcode)

### Running the Mobile App
```bash
# Get the project code
git clone <repository-url>
cd WalletGuard

# Install Flutter packages
flutter pub get

# Launch the app
flutter run
```

### Starting the Backend
```bash
# Go to the API folder
cd BudgetAPI

# Install Python packages
pip install -r requirements.txt

# Start the server
python app.py
```
